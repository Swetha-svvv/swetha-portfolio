## ⚙️ Run Locally

Clone the repository:

```bash
git clone https://github.com/Swetha-svvv/swetha-portfolio.git
```

Navigate to the project folder:

```bash
cd swetha-portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
